### Setting Production Environment

Prerequsites:
- Enable Subscription
- Install Python3
- Install AWSCLIv2
- Install Java JDK

For Red-Hat Enterprise Linux Host:
1. Install Ansible on 3 Nodes
2. Install Openshift 
3. Install odo

For Terraform: 
Deploy a EKS Cluster  

For Kubernetes Cluster:
1. Install Kubectle
2. Install Kubespary
3. Install Docker
4. Install Helm

For CD/CD:
1. Install Terraform
2. Install Jenkins
3. Install Git

For Monitoring & Metrics:
1. Install Kube-Burner
